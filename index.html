Como funciona essa Lista

Essa lista de resoluções é uma lista de resolução incremental. Ou seja, o código do exercicio 2 contém o código do exercicio 1 somado com as alterações para o exercicio 2. A mesma regra se aplica do exercicio 2 até o exercicio 7


Você notará que no primeiro exercicio não haverá nada além de HTML puro. Porém, a medida que você avança para os próximos exercicios, será possível enxergar a estilização e mais conteúdos sendo acrescentado a esse desenvolvimento WEB.
